# 🧬 Molecular Docking Utility Scripts

A comprehensive toolkit for high-throughput molecular docking workflows 
with AutoDock Vina, AutoDock-GPU, and MGLTools.

## Overview
- **Pre-Docking**: Ligand preparation, atom type detection, geometry fixes
- **Post-Docking**: Spatial filtering, affinity sorting, pose extraction
